## 🛠️ 기술 스택

### 핵심 기술
- **Unity 6000.2.9f1** - 게임 엔진
- **C# 10** - 프로그래밍 언어
- **Unity Test Framework** - 테스트 프레임워크 (NUnit)
- **DI Pattern** - ServiceLocator 기반 의존성 주입

### 구현할 시스템
1. **GridMovementSystem** - 발판 이동
2. **StageStateManager** - 스테이지 상태 관리
3. **CombatSystem** - 전투 로직
4. **SkillSystem** - 스킬 및 코스트
5. **RewardSystem** - 보상 획득
6. **TestReportSystem** - 결과 리포트
